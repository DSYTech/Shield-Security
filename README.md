# Sentinel Shield Security Website

A premium corporate website concept for a private security company.

## Overview

Sentinel Shield Security is a modern security solutions website designed to build trust and showcase professional security services including corporate security, industrial protection, CCTV monitoring, and event security.

The project focuses on creating an enterprise-level digital presence with a clean UI, responsive design, and modern animations.

## Features

- Modern responsive design
- Premium dark corporate theme
- Hero section with strong CTA
- Security services showcase
- Company statistics section
- Client testimonials
- Industries served section
- Contact inquiry form
- Smooth animations and transitions
- Mobile-friendly navigation

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

## Sections

- Home
- About Company
- Security Services
- Why Choose Us
- Industries We Serve
- Testimonials
- Contact
